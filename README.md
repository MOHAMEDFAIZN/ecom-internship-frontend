## 📌 About This Project

**E-COM** is a frontend web-based e-commerce application developed as part of a one-month internship training program at **Vinsup Infotech Pvt. Ltd.**, Madurai, Tamil Nadu. This project was assigned and guided by the company's development team to provide practical, hands-on exposure to real-world frontend development workflows.

The application demonstrates core e-commerce functionality including product listing, product management (CRUD operations), and a shopping cart — all built using vanilla **HTML, CSS, and JavaScript** without any external frameworks.

> ⚠️ **Note:** This is an internship training project, not a commercial product. It was developed under supervision at Vinsup Infotech Pvt. Ltd. as part of the academic curriculum for BCA at Kalasalingam Academy of Research and Education (KARE), Krishnankoil.

## 🗂️ Folder Structure

```
e-com/
├── index.html              # Home page — product listing
├── product-details.html    # Product details with Update/Delete options
├── add-product.html        # Add new product form
├── update-product.html     # Update existing product form
├── cart.html               # Shopping cart with quantity controls
└── README.md               # Project documentation
```

> **Note:** All CSS is embedded within each HTML file using `<style>` blocks. JavaScript logic is embedded via `<script>` tags at the bottom of each file. External dependency: Font Awesome 6.5 (via CDN).

## ✨ Features

- 🏠 **Home / Product Listing Page** — Displays product cards with image, name, brand, price, and Add to Cart button
- 📄 **Product Details Page** — Full product view with description, stock availability, and Update/Delete actions
- ➕ **Add Product** — Form to add new products with name, brand, description, price, category, stock, release date, and image upload
- ✏️ **Update Product** — Editable form pre-filled with existing product data; image preview updates dynamically
- 🗑️ **Delete Product** — Removes product from listing with a confirmation prompt
- 🛒 **Shopping Cart** — View selected items, adjust quantities (+ / −), view dynamic total, and proceed to checkout
- 📱 **Responsive Design** — Compatible across desktops, tablets, and mobile screens
- 🌙 **Dark Theme UI** — Modern dark-themed interface throughout the application
- 🔗 **Multi-page Navigation** — Clean routing between all pages

## 🚀 Getting Started

### Prerequisites
No installation or build tools required. This is a plain HTML/CSS/JavaScript project.

You need only:
- A modern web browser (Chrome, Firefox, Edge)
- A code editor (VS Code recommended) — optional, for editing

### Run Locally

**Option 1 — Open directly in browser:**
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/e-com-internship-project.git

# Navigate into the folder
cd e-com-internship-project

# Open the home page
open index.html
# Or simply double-click index.html in your file explorer
```

**Option 2 — Use VS Code Live Server (recommended):**
1. Open the project folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html` → **Open with Live Server**
4. The app will open at `http://127.0.0.1:5500`
